SAFETRACE · v1.5

Agents propose.
Humans decide.

Twelve specialised workers operate through explicit tool allowlists, data-zone ceilings, budgets, timeouts and machine-readable contracts. Every run produces a replayable receipt and every successful output remains awaiting human review.

Release invariant: zero autonomous publication, contact, referral or guilt decisions.

Default deny

Unknown tools and data-zone escalation fail before execution.

Traceable runs

Inputs, outputs, model, prompt, tools, cost and latency are hashed and recorded.

Adversarial CI

False links, missing anchors, bad attribution and harmful suggestions block release.

Responsible boundary

This is a deterministic public-source and synthetic evaluation foundation. It is not authorised for real victim, witness or restricted partner data, and it does not yet operate external model providers in production.
