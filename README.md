# Digital Democracy Studio

**Open, evidence-grounded tools that help people understand and use their rights—and hold power accountable.**

## Projects

- [SafeTrace v0.2 — Public Baseline](safetrace/)  
  Germany-first infrastructure for public-interest investigations: official-source provenance, accountability gaps, political money, arms and influence, and human-reviewed conclusions.
- [SafeVoice](https://github.com/mikelninh/safevoice)  
  Calm, practical support for people experiencing digital violence.
- [GitLaw](https://github.com/mikelninh/gitlaw)  
  Legal information grounded in the actual law.

## SafeTrace principles

> Empathy for every victim. Evidence for every claim. Due process for every accused. Accountability for every proven crime.

SafeTrace is not an accusation engine. It separates verified facts, court-established findings, official allegations, analytical red flags, unresolved gaps, and corrections.

Read the [SafeTrace methodology](safetrace/METHODOLOGY.md) and [v0.3–v1.0 roadmap](safetrace/ROADMAP.md).

Built in Berlin. Open source wherever possible.
